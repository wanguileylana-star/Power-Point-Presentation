Internet Technologies and Cloud Computing

Introduction

The internet is a global network that connects computers and devices worldwide. It allows communication, sharing of information, and many online services.

Technologies that Run Using the Internet

  • Email communication
  • Video streaming
  • E-commerce
  • Internet of Things (IoT)
  • Online gaming

What is Cloud Computing

Cloud computing is the delivery of computing services such as storage, databases, servers and software through the internet instead of using a personal computer.

Types of Cloud

  • Public Cloud – Services provided over the internet and shared by many users.
  • Private Cloud – Cloud used by one organization only.
  • Hybrid Cloud – Combination of public and private clouds.

Types of Cloud Services

  • IaaS – Infrastructure as a Service
  • PaaS – Platform as a Service
  • SaaS – Software as a Service

Application Areas of Cloud Computing

  • Education – Online learning platforms
  • Business – File sharing and collaboration
  • Healthcare – Patient record management
  • Entertainment – Streaming movies and music
  • Backup and data recovery
